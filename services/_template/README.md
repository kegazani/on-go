# Service Template

Используйте этот шаблон как основу для нового сервиса.

Минимальный состав:

```text
service-name/
  README.md
  src/
  migrations/
  deploy/
  tests/
```

Что должно быть описано в `README.md` сервиса:

1. Назначение сервиса и его границы ответственности.
2. Владение данными.
3. Внешние зависимости.
4. Входящие и исходящие контракты.
5. Способ локального запуска.
